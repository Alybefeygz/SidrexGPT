📝 Test Adımları

  1. Backend'i başlatın: python manage.py runserver
  2. Frontend'i başlatın: npm run dev
  3. Bu sayfayı açın: Browser'da test edin
  4. Widget'ı kontrol edin: Sağ alt köşede robot görünmeli
